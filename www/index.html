The intcomp R package provides tools for benchmarking various cancer gene detection algorithms that are based on integrative analysis of DNA copy number and gene expression data. The pipeline is used to quantify cancer gene prioritization performance of alternative methods based on golden-standard lists of known cancer genes in real and simulated data sets [1]. We welcome contributions to include new methods and benchmarking tests to the pipeline.

Source code is available from R-Forge. See the package vignette for installation instructions and functionality.
