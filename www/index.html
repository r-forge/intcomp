intcomp: a benchmarking pipeline for integrative cancer gene detection algorithms

The intcomp R package provides a benchmarking pipeline for quantitative comparison of cancer gene detection algorithms based on integrative analysis of DNA copy number and gene expression data. The cancer gene prioritization performance of the methods is compared with respect to golden-standard lists of known cancer genes in real and simulated data sets [1]. The comparison methods include the publicly available implementations CNAmet, DRI, edira, intCNGEan, pint/simcca, PMA, PREDA/SODEGIR, and SIM. Some algorithms include multiple variants. Contributions are welcome to include new methods, data sets and benchmarking tests to the pipeline.
